# Portfolio

This project is a 3d portfolio (three.js, r3f, react) designed to showcase my 3 main web-game projects. It could also host my resume. The app should be in the style of a "single page app".

# Overview

The app should create a 3d scene in the style of a carousel of planets orbiting the sun. The sun object should be a light source, and the background should resemble a vast, colorful field of distant, twinkling stars. Each item should carry a text label. The sun should carry the label of my name, "Paul Williams". Each of the orbiting planets, acting as the carousel items, should represent one of my projects, also with a text label specifying the name of the project.

When one of the planets is clicked, it should rotate to the front and center. There should be an additional paragraph of description text, as well as a hyperlink to the project, that is displayed for the selected, front & center planet.

## Projects (carousel items)

### Glinski

link: williapa.github.io/glinski

description: Glinski's chess is the most popular hexagonal chess variant. This web app is designed in the style of "twitch plays chess", allowing a streamer to host a game of Glinski with their chat having the ability to "vote" for their side's next move. There is also an AI option which can provide moves for either side.

### MedalVersus

link: github.com/williapa/TBS

description: MedalVersus is a turn-based strategy designed on a hexagonal board built with react, drawing inspiration from "Advanced Wars". Players can create their own custom maps, compete against an opponent in real-time (using websocket.io), and track their wins and losses across their game history.

### JS Hold'em

link: williapa.github.io/jsholdem

description: A heads-up Texas Holdem poker app built using angular.js, and nothing else! Compete against a custom A.I. to win big. Built without the use of any poker or CSS 3p libraries.
